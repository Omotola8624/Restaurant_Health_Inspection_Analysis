# Restaurant_Health_Inspection_Analysis (Food Safety Performance)
An end-to-end data analytics project leveraging SQL, Excel, and Power BI to evaluate restaurant health inspection data, food safety trends, violation patterns, and uncover actionable insights into food safety performance.

## Tools Used
- SQL (Data cleaning and querying)
- Excel (Data preparation and basic analysis)
- Power BI (Data visualization and dashboard creation)

## Objectives
- Identify restaurants with poor food safety compliance
- Analyze inspection trends over time
- Highlight high-risk categories and locations
- Provide actionable insights through dashboards

## Dataset
The dataset contains restaurant inspection records including:
- Inspection scores
- Violation types
- Restaurant locations
- Inspection dates

## Key Insights
- Most common food safety violations
- Locations with highest risk scores
- Trends in inspection performance over time

## Dashboard
Power BI dashboard was created to visualize:
- Overall compliance rate
- High-risk restaurants
- Violation breakdown
- Trend analysis

## Skills Demonstrated
- Data cleaning and transformation
- SQL querying and joins
- Data visualization (Power BI)
- Analytical thinking
